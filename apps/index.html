Cartesian Plane 2

0.0

This is a redesign of the original Cartesian Plane, featuring a swanky fullscreen UI, more optimisation, better maths support and more ease of use.

The current version is fully featured, the only improvements left are to flesh out the UI, especially the function addition/editing and plot legend UI. I also might need to improve the scale generation code.

Logon Background Changer

This is a simple Windows application that helps to change the logon background in Windows Vista/7. Tested to work on Windows 7 64-bit, and it does require administrator access (as it modifies the registry).

This might be labelled internally as "LogonBGChanger2" - for a good reason - I had to make the program again from scratch because I lost the source while configuring Git with Visual Studio. They do not play nice. Anyway, click the ">" link above to get the latest release.

Calculator

An advanced JavaScript calculator that supports proper floating-point arithmetic, shifting, mathematical variables, advanced trigonometry functions and is extensible (to a degree).

CountDown

A JavaScript countdown web app that supports URL encoded countdowns, easy input, sharing your countdowns and a timer.

A port for desktop written in C# also supports multiple countdowns and Windows PC support. You can download this port with the button below (only Windows is supported in the meanwhile).

Userscripts

These are simple scripts that can be used on a webpage or website which can change or override interaction, styling or information dynamically. They can be activated with the TamperMonkey extension for Google Chrome. With the extension installed, click the header links to install the userscript.
