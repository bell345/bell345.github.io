Logon Background Changer

This is a simple Windows application that helps to change the logon background in Windows Vista/7. Tested to work on Windows 7 64-bit, and it does require administrator access (as it modifies the registry).

Calculator

An advanced JavaScript calculator that supports proper floating-point arithmetic, shifting, mathematical variables, advanced trigonometry functions and is extensible (to a degree).

CountDown

A JavaScript countdown web app that supports URL encoded countdowns, easy input, sharing your countdowns and a timer.

A port for desktop written in C# also supports multiple countdowns and Windows PC support. You can download this port with the buttons below (only Windows is supported in the meanwhile).

Userscripts

These are simple scripts that can be used on a webpage or website which can change or override interaction, styling or information dynamically. They can be activated with the TamperMonkey extension for Google Chrome. With the extension installed, click the header links to install the userscript.
