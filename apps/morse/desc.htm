Ever wanted to learn Morse code? Or do you just want the authentic old school spy experience? Enter the Morse Code Machine! With this, you can simply type in a message on the left, and click convert to Morse code to the right! Additional features include:

  • Morse code → plain text decoding
  • Realistic Morse code timings
  • Telegraph-like Morse code input, with timing feedback
  • Encodes messages manually, in real time
  • Encodes messages automatically in an instant
  • Adjustable speed and beep frequency
