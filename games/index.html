Games

As you can see, I've been playing with web apps made with JavaScript, learning things along the way. Nothing wrong with that - it's just the way I code. But recently I have been running out of ideas for web apps. They've all been done! So I decided to experiment with games - specifically the programming of them. At the moment, my eyes are focused upon Java and various engines (Java3D) and Unity, both of which I enjoy immensely.

Lots of this stuff is not only unfinished, but never going to be finished. Again, if I do complete any of this, it'll go on the applications page.

Space Shooter

0

Using the free version of Unity Editor, I followed a basic tutorial to make a top-down space shoot-em-up, to get a feel for Unity and how it works.

About the game: WASD to move, Left-Click to fire, and R to restart once you explode. Simple and fun (to make, at least). I must note that this is at least 20 times better looking, more workable and more fun to play with than my most advanced Game Maker game. Bear in mind - I had the same budget.
