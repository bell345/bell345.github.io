WebDrawQuiz is a flexible platform for party games where answers are submitted as drawings.

The core of the game is facilitated using a WebSocket API, with auth tokens provided by a basic HTTP API. The game has been designed with both desktop computers and mobile devices in mind. It features a clean, no-frills design that is both functional and visually appealing.

For me, it was a great exercise in creating a standalone web service and resolving the challenges involved in coordinating data between the clients and the server's memory and long term storage.
