tXtEng

Version 0.2.1

After seeing the joy of command line interface games and controls, I decided to create a platform that would make it easier to develop these kind of programs for the Web. At the moment, it is in early alpha, so it's featureless. At the moment (v0.2.1), I have implemented:

  • A suitable command line interface
  • Message Of The Day system
  • Command based control protocol
  • The ability to print to the console (print command)
  • Framework for programs (run command) and a test program (flip)
  • A few other utility commands (cl, help)

NOTES:

  • Unlikely to ever be worked upon again, because it's kinda hard to make a fully functioning terminal emulator in JavaScript.

Cartesian Plane

In my maths class, I wanted to visualise the linear functions that we were studying without using a tedious pen and paper, so I created a <canvas> program to plot them for me! This started off as a small test app, but it's recently become my most worked-on project.
You can use your own functions using the console through the CrtPlane.func() method. It takes a LinearFunc, RelationFunc or regular object with an eval(x) property that spits out a y value as the parameter.
